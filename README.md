# Vintage Cam Thrift & Flip

A text-based trading game where you buy and sell vintage cameras to become wealthy enough to afford the legendary Leica 0-Series No. 105.

## Requirements

- Python 3.x
- No additional dependencies required (uses built-in `sqlite3`)

## How to Run

```bash
python main.py
```

## How to Play

1. **Start a new game** - Enter your name
2. **Go Thrifting** - Each day, browse 3 random camera/film listings
3. **Buy cameras** - Purchase items and hope for good conditions (Mint, Good, Working, Broken, Parts Only)
4. **Check Inventory** - View your purchased items
5. **Sell** - Sell as-is or repair broken cameras for more profit
6. **Win Condition** - Reach P1,000,000 to buy the legendary Leica 0-Series No. 105!

## Tips

- Film rolls are quick flips with lower risk
- Broken cameras can be repaired to sell for more
- Some cameras are rare and worth a lot
- Pass on a day to find loose change (+P50)
